# Презентация: AI Infrastructure, Mining & Data Centers

**Venture MBA Club · 3 марта 2026 · Eugene Kitkin**

## Как пользоваться

1. **Откройте презентацию:** откройте файл `index.html` в браузере (двойной клик или «Открыть с помощью» → Chrome/Safari/Firefox).
2. **Навигация:** кнопки «←» и «→» внизу экрана или стрелки клавиатуры (← →) и пробел (вперёд).
3. **Скрипт:** полный текст выступления — в `SPEAKER_SCRIPT_RUSSIAN.md`. Под каждым слайдом в презентации есть подсказка, что говорить.
4. **Слайды:** 38 штук, соответствуют скрипту. Внизу — номер слайда и прогресс-бар сверху.

## Файлы в папке

- **index.html** — презентация (открывать в браузере).
- **SPEAKER_SCRIPT_RUSSIAN.md** — полный скрипт выступления с таймингом.
- **AI_INFRASTRUCTURE_PRESENTATION_OUTLINE.md** — структура и заметки по слайдам.
- **PRESENTATION_SLIDES_DETAILED.md** — детальное содержание каждого слайда.

На слайдах указаны источники данных: McKinsey, Goldman Sachs Research, Cambridge Centre for Alternative Finance, отчёты Big Tech, TechCrunch, earnings calls, отраслевые отчёты. Цифры и графики приведены в соответствии с этими источниками.
